Cryptoverse – Cryptocurrency Dashboard

Overview- Cryptoverse is a React-based cryptocurrency dashboard that provides real-time market data, price tracking, and historical analysis through interactive charts.

Features- Live cryptocurrency price tracking Market cap & volume insights 5-year historical data visualization Search and filter functionality Interactive charts (Chart.js) Responsive and user-friendly UI

Tech Stack- React.js, HTML, CSS Chart.js, Styled Components React Router RapidAPI (crypto data)

Working- Fetches real-time cryptocurrency data from APIs and displays it using dynamic components and visual charts for easy analysis.
